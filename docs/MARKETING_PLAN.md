## Stratégie Meta Ads (150€ budget total)

### Structure des campagnes

**Campagne 1 — Trafic / Notoriété (50€)**
- Objectif : Trafic vers la boutique
- Audience : Femmes et hommes 25-45 ans France, intérêts : gadgets, maison connectée, Amazon, AliExpress, dropshipping
- Budget : 5€/jour × 10 jours
- Visuels : Carrousel produits (3-4 produits vedettes)

**Campagne 2 — Conversion / Achat (70€)**
- Objectif : Conversions (événement Achat)
- Audience : Lookalike des visiteurs du site (1%)
- Nécessite le Pixel Meta installé sur vellio.fr
- Budget : 10€/jour × 7 jours
- Visuels : Vidéo TikTok recyclée + texte avec prix et garantie

**Campagne 3 — Retargeting (30€)**
- Objectif : Conversions
- Audience : Visiteurs du site sans achat (30 derniers jours)
- Budget : 5€/jour × 6 jours
- Visuels : Image statique avec code promo -10%

### Installation Pixel Meta
1. Aller dans Meta Business Suite > Gestion des événements
2. Créer un pixel > Copier l'ID
3. Ajouter dans `next.config.js` ou via `<Script>` dans `layout.tsx`
4. Événements clés à tracer : `PageView`, `AddToCart`, `InitiateCheckout`, `Purchase`

### Créatifs qui convertissent
- Fond blanc + produit centré + prix en rouge
- Avant/Après (problème → solution)
- Badge "TikTok viral" + score tendance
- Garantie 30j visible en bas de l'image
- Livraison gratuite mise en avant si applicable
